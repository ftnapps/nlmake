NLMake

Debian packaging for NLMake

This project is primarily about Debian packaging for NLMake, which is an open-sourced FTN/Fidonet nodelist processing program. It is used to process FTN/Fidonet nodelists for submission to coordinators and/or for the creation of nodelist files for distribution.

  • The debian branch in the repository is used for the development of the debian packaging directory for NLMake. It includes the code merged from the master branch.
  • The master branch in the repository contains the code from the NLMake application itself.
